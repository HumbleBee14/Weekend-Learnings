# SQL - Practice

Practicing SQL queries and database concepts.

## Setup

```bash
# Option 1: SQLite (zero setup)
sqlite3 practice.db

# Option 2: PostgreSQL via Docker
docker run --name pg-practice -e POSTGRES_PASSWORD=practice -p 5432:5432 -d postgres
```

## Exercises

| # | Exercise | Description |
|---|----------|-------------|
| 01 | basic-queries | SELECT, WHERE, ORDER BY, LIMIT |
