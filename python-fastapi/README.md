# Python - FastAPI

Learning FastAPI from scratch through hands-on exercises.

## Setup

```bash
cd python-fastapi
python -m venv venv
source venv/bin/activate  # or venv\Scripts\activate on Windows
pip install fastapi uvicorn
```
